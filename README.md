# 🧠 AI Анализатор тональности текста (Fullstack проект)

## 📌 Описание проекта
Это fullstack веб-приложение с интеграцией искусственного интеллекта, которое анализирует текст пользователя и определяет его тональность:  
**позитивная, негативная или нейтральная**.

Результат анализа сохраняется в базе данных и отображается в истории.

---

## 🚀 Возможности проекта
- Анализ текста с помощью AI/ML модели
- Определение тональности текста
- Вывод confidence (уверенности модели)
- История всех запросов
- Очистка истории
- График статистики результатов
- Полная fullstack архитектура

---

## 🏗 Архитектура проекта

Пользователь → Frontend → Backend (Flask) → AI модель → База данных → Frontend

---

## 🤖 Искусственный интеллект (AI)

- Алгоритм: Logistic Regression
- Метод преобразования текста: TF-IDF
- Библиотека: scikit-learn

### Как работает модель:
1. Текст пользователя преобразуется в числовые признаки (TF-IDF)
2. Модель классифицирует текст
3. Возвращает:
   - класс (positive / negative / neutral)
   - вероятность (confidence)

---

## 💾 База данных

Используется SQLite.

### Таблица результатов:
- id — уникальный номер
- input_text — введённый текст
- ai_result — результат AI
- confidence — уверенность модели
- created_at — время запроса

---

## ⚙️ Backend (Flask API)

### Основные endpoints:

**POST /analyze**
- принимает текст
- возвращает результат AI

**GET /results**
- возвращает историю анализа

**POST /clear**
- очищает историю
